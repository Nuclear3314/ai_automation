# 项目名称

## 简介
这是一个简单的 C 项目示例，演示如何编译和运行程序。

## 文件结构
```
D:\
└── ai_automation_project
    ├── .github
    │   └── workflows
    │       └── ci.yml
    ├── Makefile
    ├── main.c
    ├── hello.c
    └── hello.h
```

## 编译和运行

### 在 MSYS2 或 PowerShell 中

1. **编译项目**：
    ```sh
    make
    ```

2. **运行可执行文件**：
    ```sh
    ./main.exe
    ```

3. **清理项目文件**：
    ```sh
    make clean-win
    ```
